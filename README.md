# Emergency Recorder

Лендинг-страница для Android-приложения **Emergency Recorder** — приложения для быстрой аудиозаписи в экстренных ситуациях с параллельной выгрузкой в облачные сервисы пользователя.

🌐 **Сайт:** https://stasionok.github.io/emergency-recorder/

## Содержимое

- `index.html` — главная страница с описанием приложения, Privacy Policy и Terms of Service
- `style.css` — стили

## Назначение

Эта страница используется как обязательный публичный URL при регистрации OAuth-приложений у облачных провайдеров:

- Google Drive (Google Cloud Console)
- Dropbox (App Console)
- Yandex.Disk (oauth.yandex.ru)
- pCloud (My Apps)
- Box (Developer Console)
